Think and Code

TESTING VERSION

Think and Code adalah game edukasi di mana pemain diajak untuk berpikir logis terlebih dahulu, lalu menuliskan kode sederhana untuk menggerakkan karakter dan menyelesaikan misi. Di game ini, pemain belajar memahami cara kerja pemrograman dasar seperti perintah gerak, pengulangan, dan logika, sambil memecahkan tantangan secara bertahap. Tujuannya bukan hanya menang, tapi melatih cara berpikir terstruktur, efisien, dan kreatif lewat kode.

Versi ini masih dalam masa pengujian (testing). Beberapa fitur mungkin belum stabil, dan kami sangat terbuka terhadap masukan.
